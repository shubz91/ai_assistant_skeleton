## Rules

1. **Limit fonts**: Use 2-3 fonts maximum — display, body, and optionally mono
2. **Body text**: Use `#444444` (not pure black) for body copy
3. **Numbers in mono**: All numeric values, stats, and code in the monospace font
4. **Line height**: Body text at 1.7-1.8 line-height for comfortable reading
5. **Max line length**: 60-75 characters per line for readability
