## Key Rules
1. **Replace placeholder values** — update all `[PLACEHOLDER]` values in detail files with your actual brand assets
2. **Warm backgrounds** — avoid pure white; prefer off-white or light tinted backgrounds
3. **Consistent fonts** — establish a 2-3 font system and never deviate from it
4. **Single accent per composition** — limit accent color usage to one per view
5. **Numbers in monospace** — use a monospace font for all stats, percentages, and data values
6. **Voice consistency** — refer to voice-tone.md before writing any copy
